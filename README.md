# Web Resume Labs 🚀

## Description 📝

This project is a personal resume website created for the "Web Application Development" course at VSTU Borisoglebsk. It builds step by step: HTML structure, CSS styling (with Win8-inspired tiles 😎), JS interactivity (forms, validation, modals, events, video playback 🎥).

Features labs 1-4: frames, lists/tables, form validation, alerts/confirms/prompts, event handlers (onload/onclick/onsubmit), and post-registration video window.

## Features ✨

- Lab 1: Basic HTML tags, frames, tables, lists, images/links.
- Lab 2: CSS methods (inline/embedded/external), selectors, pseudo-classes, tile design.
- Lab 3: JS form validation (login/password/email), developer info modal.
- Lab 4: JS windows (alert/confirm/prompt), 3+ events, video in modal with pause/resume.

## Structure 📂

- `Main.html`: Frames entry (header/resume/greeting).
- `Header.html`: Title.
- `Resume.html`: Resume sections in tiles.
- `Greeting.html`: Welcome + registration form + events/modals.
- `Additional*.html`: Extra pages.
- `styles.css`: All styles.
- `script.js`: JS logic.
- Images: avatar.jpg, more.jpg.
- Video: video.mp4 (replace with yours).

## Setup & Usage 🛠️

1. Clone: `git clone https://github.com/Nipplexxx/web-resume-labs.git`  
2. Open `Main.html` in browser.  
3. Interact: Fill form, click buttons for modals/events/video 🎉.

No server needed—just local files!

## Technologies 🧰

- HTML5  
- CSS3 (grid/transitions)  
- JavaScript (DOM/events/RegExp/window)
